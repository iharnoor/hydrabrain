# Project Alpha — HydraBrain Demo

Project Alpha is a personal knowledge brain built on HydraDB.
It supports hybrid search combining vector similarity, BM25 keywords, and graph traversal.

## Key Features
- **Capture**: ingest thoughts and consumed content instantly
- **Sync**: bulk incremental import of markdown directories
- **Think**: synthesized answers with citations (requires Gemini)
- **Graph**: explore entity relationships in the knowledge graph
